# Springer AI&L Submission Checklist

Journal: Artificial Intelligence and Law (Springer)
Review mode: Double-anonymous

---

## Required files for submission

| File | Status | Path |
|---|---|---|
| Anonymized manuscript (.docx) | TODO: convert from .md | `submission/springer_ailaw/anonymized_manuscript/LEXON_AILaw_anonymized.md` → `.docx` |
| Separate title page (.docx) | TODO: convert from .md | `submission/springer_ailaw/title_page/title_page.md` → `.docx` |
| Anonymized review artifact (ZIP) | Generated by script | `submission/springer_ailaw/anonymous_artifact/lexon_anonymous_review_artifact.zip` |
| Figure source file | Check existence | `outputs/figures/pipeline.png` or `Fig1.png/.eps/.tiff` |
| Cover letter (optional) | Manual | — |
| Declarations | Enter in submission interface | See `title_page.md` |

**Do not submit only a PDF.** Springer requires editable source files.

---

## Automated checks

Run all checks with:

```bash
make ailaw-submission-check
```

Individual checks:

```bash
# Build anonymous artifact
make anonymous-artifact

# Blind check (must pass with no forbidden identifiers)
make blind-check

# Springer style checks
make springer-style-check

# Full pipeline
make validate
make traces
make validate-mapping
```

## Checklist items

### Manuscript

- [ ] Author name removed
- [ ] Affiliation removed
- [ ] Email removed
- [ ] "Submitted to Artificial Intelligence and Law" line removed
- [ ] Declarations section removed (moved to title page / submission interface)
- [ ] Data availability replaced with anonymous version
- [ ] GitHub/Zenodo links replaced with "[anonymized review artifact supplied as supplementary material]"
- [ ] AffectLog references removed
- [ ] Citations converted from numeric [N] to author-year style
- [ ] Reference list alphabetized
- [ ] Keywords reduced to 6
- [ ] "Figure 1" changed to "Fig. 1"
- [ ] Abstract 150–250 words
- [ ] Tables numbered consecutively in Arabic numerals
- [ ] All tables cited in text

### Artifact

- [ ] `make reproduce` runs cleanly
- [ ] `make ailaw-artifact-check` passes
- [ ] Blind check passes (no forbidden identifiers in anonymous package)
- [ ] Anonymous artifact ZIP generated

### Submission interface

- [ ] Author name and affiliation entered in submission system
- [ ] Competing interests declaration entered
- [ ] Funding statement entered
- [ ] AI use disclosure entered
- [ ] Ethics statement entered
